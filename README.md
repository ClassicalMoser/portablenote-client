## Usage

```bash
npm install
```

### Learn more on the [Solid Website](https://solidjs.com) and [Tauri](https://tauri.app).

## Available Scripts

### Web (Vite)

- **`npm run dev`** – Dev server at [http://localhost:5173](http://localhost:5173)
- **`npm run build`** – Production build to `dist/`
- **`npm run preview`** – Preview production build

### Tauri (desktop)

- **`npm run tauri:dev`** – Run the app in development (starts Vite + Tauri window)
- **`npm run tauri:build`** – Build the desktop app for production
- **`npm run tauri`** – Run other Tauri CLI commands (e.g. `npm run tauri -- info`)
